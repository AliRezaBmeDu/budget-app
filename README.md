# 📖 My-Blog <a name="about-project"></a>

**Budget-App** is a Ruby-On-Rails built application which is a classic example of a budget application where users can store their transactions under the categories. The data storage is taken care of by Postgresql database.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

### Setup

Clone this repository to your desired folder:

https://github.com/AliRezaBmeDu/budget-app.git

### Install

```sh
  cd budget-app
  bundle install
  npm install
```

### Usage

To run the project, execute the following steps:

```sh
  rails server
```
Then navigate to 'https://localhost:3000' to get started

### Test

To run the unit test files, run the following command in the terminal:

```sh
  rspec
```
